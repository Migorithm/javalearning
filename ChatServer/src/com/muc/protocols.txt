User & Server

1. User --> Server
    *login //logoff
    *status

2. Server --> User
    *online / offline


3. User --> User
    *Direct messages
    *Broadcast messages / group messaging


Commands:

    login<user><password>
    logoff

    msg <user> body....
    i.e. guest : "msg jim Hello World" -> jim will receive a message
         jim  : "msg guest Hello World"

    msg #topic body
